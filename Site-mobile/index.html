Mude a Orientação do seu Dispositivo

Podemos configurar os parâmetros de Media Type e Media Features das nossas Medias Queries.

    • Media Type se refete tipo de tela, pode ser screen, print ou all, que engobla as outras duas.
      Media Features são características dessa tela que podemos especificar para receber um dado conjunto de CSS.

    • Sendo assim, podemos ter arquivos CSS separados para telas e impressoras, e dentro de telas, um para modo retrato e outro para modo paisagem do celular, por exemplo. Basta criar os arquivos CSS correspondentes.
